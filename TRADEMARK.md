# Trademark Policy

> **Draft notice:** like the CLA, this policy is a starting point and has not been reviewed by a lawyer. Have it reviewed together with the CLA, and consider formally registering the mark.

## The Marks

"**Traverse**", "**Traverse Framework**", the `traverse-framework` organization name, and any associated logos (the "Marks") identify this project and its owner. The Marks are owned by the project owner identified in [GOVERNANCE.md](GOVERNANCE.md).

## What the Code License Does Not Cover

All code in this organization is available under an open source license (Apache-2.0 unless a repo states otherwise). **That license does not grant any trademark rights.** Section 6 of the Apache License 2.0 states this explicitly. You can use, modify, fork, and redistribute the code freely — but the right to call the result "Traverse" is separate.

## Allowed Without Permission

- Referring to the project by name to truthfully describe it ("built with Traverse", "compatible with the Traverse Framework").
- Unmodified redistribution of official releases, keeping the original name and attribution.
- Package names or module paths that reference the project as a dependency identifier (e.g. `traverse-client-go`), provided they don't imply official status.

## Requires Permission

- Using the Marks in the name of a fork, derived product, company, service, or domain in a way that suggests it is the official project or is endorsed by the project owner.
- Using the Marks in logos or branding for commercial offerings.

Ask by opening an issue in this repository or contacting the owner.

## Forks

Forks are welcome and expected under the open source license. A public fork that materially diverges from this project must be renamed so users are not confused about which is the official Traverse.
