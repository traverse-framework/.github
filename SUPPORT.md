# Support

Thanks for using Traverse. Here's where to go for help:

- **Bug reports / feature requests** — open an issue in the affected repository using the issue templates. Templates are shared org-wide from this repo.
- **Questions and discussion** — open an issue with the question label in the relevant repo. If GitHub Discussions is enabled on a repo, prefer it for open-ended questions.
- **Security issues** — never open a public issue; follow [SECURITY.md](SECURITY.md).
- **Contributing** — read [CONTRIBUTING.md](CONTRIBUTING.md) first; contributions require accepting the [CLA](CLA.md).
- **Governance / licensing / trademark questions** — open an issue in this (`.github`) repository.

This is an open source project maintained on a best-effort basis; there is no commercial support or SLA at this time.
